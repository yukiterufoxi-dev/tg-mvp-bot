# Karta Clean — Stage 1 (структура репозитория)

Этот пакет содержит пустую структуру папок для проекта:
- `bot/` — код Telegram-бота (aiogram 3.x)
- `web/` — FastAPI (карта/админка)
- `db/` — модели/запросы БД
- `services/` — общие сервисы (SMTP, storage, crypto)
- `ops/` — деплой/сервисы
- `data/` — медиа/БД (в гите не храним файлы, только .gitkeep)
- `docs/` — документация

Загрузите содержимое в ветку `dev` вашего репозитория.
